# Security Policy

## Supported versions

The current `main` branch is supported.

## Data handling

This project is a synthetic portfolio prototype. It must not contain:

- VINs, vehicle identifiers, device serials, plate numbers, or customer identifiers.
- Precise GPS traces, route history, depot coordinates, or driver/operator names.
- Production autonomy logs, incident reports, safety cases, or proprietary fleet data.
- Cloud credentials, API tokens, SSH keys, private certificates, or deploy secrets.

## Reporting

Open a GitHub issue for security concerns that do not expose sensitive data. For sensitive issues, contact the repository owner privately.
